LearnX AI - deployment-ready package

Instructions:
1. Create new GitHub repo and push these files.
2. On Render, connect the repo and deploy (Render will run install and start).
3. Set environment variable OPENROUTER_KEY with your OpenRouter API key in Render dashboard.
4. Visit /full or /lite to see respective UI.

Notes:
- SQLite database file `database.db` is created on first run.
- Do not commit your secret keys.
